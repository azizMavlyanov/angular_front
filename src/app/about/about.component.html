О нас

“Голос Студента” - это учебный проект, представляющей собой электронный блог для размещения статей. Данный сервис позволяет студентам разных университетов размещать статьи научного характера.

Студентам для размещения статьи в данном блоге необходимо зарегистрироваться, указав электронную почту, имя и пароль. Далее студентам необходимо верифицировать свою электронную почту. Для этого необходимо проверить свою, на которую будет отправлена ссылка для верификации почты студентов. После верификации электронной почты студентам необходимо перейти на страницу “Войти”. На данной странице необходимо заполнить форму, указав электронную почту и пароль, и нажав на кнопку войти. После входа в систему у студентов появляются права на размещения своих статей в данном блоге. Буду авторизованном пользователями, студенты могут размещать свои статьи в данном блоге. Для этого студентам необходимо перейти на главную страницу и нажат на кнопку “Опубликовать”. После нажатия на данной кнопку студент будет перенаправлен на страницу формы для введения данных о публикуемой статье. После введения соответствующих данных необходимо нажать на кнопку отправить. Таким образом будет опубликована любая статья. Опубликованную статью можно найти в писки опубликованных статей на странице “Статьи”.
